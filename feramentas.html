Ferramentas W3

Essas ferramentas auxiliam os gerenciamentos de servidores W3, geração de hipertexto, etc. Daemon básico W3 Parte do código do projeto W3. Servidores gateway que você pode pegar e adaptar. Interface framemaker Existem alguns arquivos de piche no arquivo FTP anônimo no file://info.cern.ch/www/src que permitem que o FRAMEmaker seja usado como uma ferramenta W3. Dan Conolly, Convex. Incldues MIF tradução HTML. Transformando HTML em TeX Fizemos isso com o "WWW Book" para imprimi-lo. Consulte o Makefile, por exemplo, e os scripts html2latex.sed e sub1.sed. Nós escrevemos uma introdução especial, mas de outra forma todo o texto era hipertexto do projeto W3. Gerando HTML Estes são scripts para gerar hipertexto SGML a partir de coisas como listagens de diretórios, etc. Além disso, para verificar e corrigir HTML duvidoso. Análise de log de servidor Analisar registros de servidores requer, antes de mais nada, alterar os numédicos numédicos numédicos do nó da internet em nomes de domínio. httpd-analysis.c é um programa para fazer isso. Alimente os resultados através de awk e grep de sua escolha!
