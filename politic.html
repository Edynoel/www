Política

Isso descreve a política do projeto W3 no CERN. Embora não seja legalmente vinculativo, isso tenta explicar meu entendimento das regras do CERN e dos desejos da equipe do CERN.

Mirar

O objetivo básico do projeto é promover a comunicação e a disponibilidade de informações para a comunidade de Física de Alta Energia (HEP). O projeto é baseado no CERN, cujo orçamento é fornecido por contribuições do dinheiro do contribuinte dos Estados-membros europeus. É do interesse do HEP, do CERN, e do próprio projeto que ele deve intertrabalho com sistemas e informações em muitos outros campos, e por isso a colaboração ativa com outros grupos é essencial. Produzir um sistema de informação isolando o HEP do resto do mundo seria contraprodutivo, de modo que o objetivo pode ser visto como promover uma rede global de informações. A equipe www está entusiasmada que informações de todos os tipos devem estar disponíveis da forma mais ampla possível.

Colaboração

Incentivamos a colaboração por parte de partes acadêmicas ou comerciais. Há sempre muitas coisas a serem feitas, portas a serem feitas para diferentes ambientes, novos navegadores a serem escritos e dados adicionais a serem incorporados na "web". Já houve muitas contribuições nestes termos, e também com o suporte de hardware dos fabricantes. Se você estiver interessado em estender a web ou o software, por favor, envie um e-mail ou telefone para nós.

Distribuição de código

O código escrito no CERN é coberto pelos direitos autorais do CERN. Na prática, a interpretação disso no caso do projeto W3 é que os programas estão livremente disponíveis para os órgãos acadêmicos. Para organizações comerciais que não estão revendendo, mas estão usando-a para participar da troca global de informações, a cobrança é geralmente dispensada para reduzir os custos administrativos. O código é, naturalmente, compartilhado livremente com todos os colaboradores. As organizações comerciais que desejam vender software com base no código W3 devem entrar em contato com o CERN. Estamos em processo de obtenção de acordo para liberar certas partes do código do projeto WWW com a Licença Pública Geral (e licença de biblioteca GP). Quando o código CERN é incluído no código de domínio público, esse código CERN torna-se também domínio público. O código não originário do CERN é, naturalmente, coberto por termos definidos pelo titular dos direitos autorais envolvido.

Protocolos e Formatos de Dados

A definição de protocolos como HTTP e formatos de dados como HTML são de domínio público e podem ser usados livremente por qualquer pessoa.
